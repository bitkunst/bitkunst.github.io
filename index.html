Block Explorer 사이트 제작

 Github 
경일게임아카데미

Geth를 사용해 private 블록체인 네트워크를 구축하였으며 web3 라이브러리를 사용해 블록 정보, 계정 정보, 트랜잭션 정보 등을 실시간으로 조회할 수 있는 블록 익스플로러 사이트를 제작하였습니다.

기술 스택 : Geth, web3, React, Node.js
주요 기능 : private 블록체인 네트워크를 구축하였으며 웹소켓을 사용하여 블록이 마이닝되거나 트랜잭션이 블록에 추가될 때 실시간으로 해당 정보를 업데이트하여 사이트에서 조회 가능. web3 라이브러리를 사용해 작업하였으며 계정 주소로 검색시 해당 계정으로 마이닝한 블록 내용 조회, 해당 계정으로 발생시킨 트랜잭션 조회 가능. 블록 Number로 검색시 해당 블록에 대한 정보 조회 가능. 트랜잭션 Hash로 검색시 해당 트랜잭션에 대한 정보 조회 가능.
담당 역할 : 개인 프로젝트
